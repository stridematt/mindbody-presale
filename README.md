# Mindbody Presale

Serverless presale system built with Next.js, Vercel, Neon PostgreSQL, and the Mindbody Public API v6.

## Stack

- Next.js App Router
- Vercel
- Neon PostgreSQL
- Mindbody Public API v6

## Environment Variables

Set these in Vercel project settings before deploying:

```bash
MINDBODY_API_KEY=
MINDBODY_SITE_ID=
MINDBODY_USER_TOKEN=
DATABASE_URL=
NEXT_PUBLIC_APP_URL=
